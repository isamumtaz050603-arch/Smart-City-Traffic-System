# Smart-City-Traffic-System
A Python-based Smart City Traffic Simulation built using the Tkinter GUI library. This project simulates a smart traffic management system where vehicles move dynamically across a city road network. Users can spawn normal and emergency vehicles, visualize their movement between intersections, and monitor active traffic in real time.
1)Features
   Interactive smart city map with connected roads and intersections
   Dynamic vehicle spawning and movement simulation
   Emergency vehicle support with priority visualization
   Real-time active vehicle tracking panel
   Graphical user interface built with Tkinter
   Pathfinding between city nodes
   Smooth animated traffic movement
2)Technologies Used
   Python
   Tkinter
   Object-Oriented Programming (OOP)
   Mathematics for movement calculations
3)Use Case
   This project demonstrates the basic concepts of:
    Smart city infrastructure
    Traffic flow simulation
    GUI application development
    Vehicle routing systems
4)How to Run
    Install Python 3
    Run the file:
      python smart_city_Project.py
